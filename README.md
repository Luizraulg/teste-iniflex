# Teste Prático - Iniflex

Projeto desenvolvido em Java para o teste prático da Projedata.

## Funcionalidades implementadas

- Cadastro de funcionários
- Remoção de funcionário
- Formatação de datas e salários
- Aumento salarial
- Agrupamento por função
- Ordenação alfabética
- Cálculo de salários mínimos
- Funcionário com maior idade
- Total dos salários

## Tecnologias utilizadas

- Java 21
- Eclipse IDE

## Como executar

1. Importar o projeto na IDE Eclipse.
2. Executar a classe `Principal.java`.

## Estrutura do projeto

- Pessoa.java
- Funcionario.java
- Principal.java
